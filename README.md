## Description

:facepunch: An easier Highlight in Vue.js :facepunch:

## Project setup

```
yarn add vuejs-highlight

npm i vuejs-highlight
```

```
<script>
  import Highlight from 'vuejs-highlight';
  Vue.use(Highlight);

  <vue-highlight></vue-highlight>
</script>
```
